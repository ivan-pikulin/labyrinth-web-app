# Labyrinth — UI Концепция

## Эстетическое направление

### «Каменные Руны»
**Ancient Stone Tablet** — интерфейс как высеченные на камне руны, которые оживают при активации. Ощущение древнего подземелья, где игрок читает послания на стенах.

### 3.2 Кнопки направлений

**Неактивная (стена):**
```
┌───────┐
│ ░░░░░ │  Серая, приглушённая
│ ░ ← ░ │  Стрелка едва видна
│ ░░░░░ │
└───────┘
```

**Активная (проход):**
```
┌───────┐
│ ▒▒▒▒▒ │  Светлее фона
│ ▒ ← ▒ │  Чёткая стрелка
│ ▒▒▒▒▒ │
└───────┘
```

**При наведении/нажатии:**
```
╔═══════╗
║ ▓▓▓▓▓ ║  Золотая рамка
║ ▓ ← ▓ ║  Лёгкое свечение
║ ▓▓▓▓▓ ║
╚═══════╝
```

### 3.3 Режимы действий

```
Активный режим:
╔══════╗  ┌──────┐  ┌──────┐  ┌──────┐
║ 🚶   ║  │ 🏹   │  │ 💣   │  │ 🧱   │
║ Идти ║  │   5  │  │   2  │  │   —  │
║  ✓   ║  │  ⇧   │  │  ⌃   │  │  ⌥   │
╚══════╝  └──────┘  └──────┘  └──────┘
 золотой   обычный   обычный  недоступен
```

**Индикаторы горячих клавиш:**
- `⇧` = Shift
- `⌃` = Ctrl
- `⌥` = Alt

### 3.4 Статус-бар

```
♥ ▰▰▰▰▱  4/5     ➴ 5     ◉ 2     ⬡ 🛡
  здоровье       стрелы   бомбы   инвентарь
```

**Полоска здоровья:**
- Заполненные сегменты `▰` — текущее HP
- Пустые сегменты `▱` — потерянное HP
- Цвет меняется: зелёный → жёлтый → красный

## 6. Анимации и эффекты

### 6.1 Микро-анимации

| Событие | Эффект |
|---------|--------|
| Нажатие кнопки | Лёгкое вдавливание + золотая вспышка |
| Смена режима | Плавное перетекание подсветки |
| Новое сообщение | Fade-in снизу |
| Ощущение опасности | Пульсация иконки (🔥 мерцает) |

### 6.2 Макро-анимации

| Событие | Эффект |
|---------|--------|
| Получение урона | Красная вспышка по краям экрана |
| Смерть | Fade to black → текст → fade in на кладбище |
| Победа | Золотое сияние, частицы |
| Телепортация | Искажение/рябь экрана |

### 6.3 Ambient

- Лёгкие частицы пыли в воздухе (очень subtle)
- Мерцание факелов (тени по краям)
- При idle — изредка "капли воды" в логе

## 8. Адаптивность

### Desktop
- Полный интерфейс с логом, статус-баром, кнопками
- Горячие клавиши (стрелки + модификаторы)
- Hover-эффекты на кнопках

### Mobile / Touch
- Компактный статус-бар
- Крупные touch-friendly кнопки направлений
- Swipe-жесты для направлений (опционально)
- Режимы действий в нижней панели
